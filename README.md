# Intelligent-front-desk-frame-work-for-enhancing-patient-flow-in-hospitals-
📌 Overview
The Intelligent Front Desk Framework is an AI-powered system designed to automate patient registration and improve hospital front desk efficiency. It replaces manual processes with a smart triage system that collects patient details through text or speech, analyzes symptoms, and directs patients to the appropriate department.
The system generates a unique token for each patient and sends it to their mobile device for seamless queue management. Additionally, it provides navigation assistance, guiding patients to the allocated department within the hospital.

✨ Features
🤖 AI-Powered Smart Triage (Text & Speech Input)
🏥 Automatic Department Allocation
🎟️ Token Generation & Mobile Notification
🧭 Department Navigation Guidance inside hospital
🔥 Real-Time Database using Firebase
🌐 Hospital Management Website
📊 Efficient Patient Flow Management

⚙️ How It Works
Patient enters name, age, gender, and symptoms via text or speech
AI performs smart triage and analyzes symptoms
System assigns the appropriate department
A unique token is generated
Token is sent to the patient’s mobile device
System provides navigation guidance to reach the department
Data is stored in Firebase for real-time access
Admin monitors via the hospital management dashboard

🎯 Key Highlight
This system not only automates registration and triage but also guides patients inside the hospital, reducing confusion and improving overall patient experience.
